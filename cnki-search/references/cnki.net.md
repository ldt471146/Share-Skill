## 平台特征

- **架构**：知网 kns8s 是基于 Vue.js 的 SPA 应用，页面切换通过前端路由完成
- **入口域名**：`www.cnki.net`（首页）、`kns.cnki.net`（检索系统）、`kns8s` 为新版检索入口
- **反爬行为**：
  - 短时间内高频请求会触发验证码（滑块验证）
  - 同一 IP 大量并发 tab 可能被限流
  - 自动化操作（如快速填表、跳过等待直接提交）更容易被检测
- **登录需求**：基本检索和元数据查看无需登录；全文下载、导出引用格式、部分统计功能需要机构 IP 或个人账号登录
- **内容加载**：搜索结果通过 AJAX 异步加载，需等待 DOM 更新后再提取

## 有效模式

- **高级检索入口**：`https://kns.cnki.net/kns8s/AdvSearch` 是最稳定的检索入口
- **简单检索**：`https://kns.cnki.net/kns8s/search` 可用于快速关键词搜索
- **论文详情**：从检索结果中提取的链接直接打开，不要手动构造详情页 URL
- **GUI 交互更安全**：填写搜索框 → 点击搜索按钮的 GUI 路径比直接构造 URL 参数更不容易被检测
- **操作节奏**：每次操作间隔 1-2 秒模拟人类行为

## 已知陷阱

- **URL 参数依赖**（2026-04-13）：知网检索结果的论文详情链接包含会话相关参数（如 `v=`、`QueryID=`），手动构造 URL 访问论文详情页经常返回错误页面
- **选择器不稳定**（2026-04-13）：知网前端更新频繁，DOM 选择器可能随版本变化。操作前必须先 eval 探查实际 DOM 结构
- **弹窗干扰**（2026-04-13）：首次访问可能出现 cookie 同意弹窗、登录推荐弹窗，需先处理再继续操作
- **翻页异步**（2026-04-13）：点击翻页后内容通过 AJAX 更新，必须等待新内容加载完成再提取
- **编码问题**（2026-04-13）：知网部分页面使用 GBK 编码，通过 CDP eval 获取的中文文本通常不受影响，但 curl 直接请求时可能出现乱码
